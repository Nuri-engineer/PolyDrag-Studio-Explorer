Справка по приложению PolyDrag Studio Explorer

Как использовать:

  • Нажмите кнопку Создать для генерации новых уникальных многоугольников.
  • Перетаскивайте многоугольники из левой зоны (buffer) в правую (work) для работы с ними.
  • Вы можете перемещать многоугольники внутри рабочей зоны, используя drag & drop.
  • Используйте колесо мыши для масштабирования рабочей области.
  • Зажмите левую кнопку мыши и двигайте курсор для панорамирования рабочей области.
  • Нажмите кнопку Сохранить, чтобы сохранить текущее состояние в браузере.
  • Кнопка Сбросить очистит сохранённые данные и зоны.

Горячие клавиши:

  • Esc — закрыть окно справки.
  • Enter или Пробел — открыть окно справки при фокусе на иконке ?.
